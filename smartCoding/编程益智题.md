# 编程益智题
### reduce实现map
根据原数组建立新数组并返回，初始数组是acc并有reduce的第二个参数传入
```js
//map的实现
var arr = [1,2,3];
arr = arr.map(x=>x*2);
console.log(arr);//[2,4,6];

//reduce的实现
reduce(()=>{},[])//第二个参数默认是acc的初始值
let fn = (x)=>{return x * 2};
//实现1
arr = arr.reduce((acc,curr,idx,arr)=>{acc.push(fn(curr));return acc},[])
//实现2
arr = arr.reduce((acc,curr,idx,arr)=>return [..acc,fn(curr)],[]);
```
