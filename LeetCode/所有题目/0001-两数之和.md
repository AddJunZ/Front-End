## 两数之和[leetcode-1](https://leetcode-cn.com/problems/two-sum/)

### 解法一：使用Map记录遍历过的数字避免重复，一次遍历
```js
var twoSum = function(nums, target) {
  let map = new Map();
  for(let i = 0; i < nums.length; i++){
    let cha = target - nums[i];
    if(map.has(cha)){
      return [map.get(cha), i];
    }
    map.set(nums[i], i)
  }
  return [-1,-1]
};
```
