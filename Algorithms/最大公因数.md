## 最大公因数
1. 埃拉托色尼筛选法
```js
function Sieve(n) {
  var arr = [];
  for(let i = 2; i < n; i++){
    arr[i] = i;
  }
  for(let i = 2; i < Math.floor(Math.sqrt(n)); i++){
    //如果该项没有被前面消去
    if(arr[i]){
      //这里第一个考虑的数字就是平方是因为对于一个数字k，小于k*k的数字，如k*2等会在前面就已经被约掉，因此没有必要考虑k*k之前的数
      let j = i**2;
      while(j <= n){
        arr[j] = null;
        j += i;
      }
    }
  }
  return arr.filter(x=>x);
}
console.log(Sieve(10));
//[2, 3, 5, 7, 9]
```
